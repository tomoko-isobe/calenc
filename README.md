# Calenc

Calencは曜日付きの日付リストを作成するツールです。

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'calenc'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install calenc

## Usage

```bash
$ calenc
```
